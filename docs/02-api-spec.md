# API 명세서

## 1. 전체 다짐 목록 조회

| 항목         | 내용                                                                                       |
| :----------- | :----------------------------------------------------------------------------------------- |
| **URL**      | `GET /api/guestbooks`                                                                      |
| **설명**     | 모든 다짐 목록을 **작성일 최신순**으로 조회.                                               |
| **Request**  | 없음                                                                                       |
| **Response** | `[{ "id": 1, "author": "작성자", "content": "내용", "createdAt": "2025-12-17T..." }, ...]` |

## 2. 다짐 등록

| 항목         | 내용                                                                                |
| :----------- | :---------------------------------------------------------------------------------- |
| **URL**      | `POST /api/guestbooks`                                                              |
| **설명**     | 새로운 다짐을 등록. (작성일은 서버 자동 생성)                                       |
| **Request**  | `{ "author": "작성자", "content": "내용" }`                                         |
| **Response** | `{ "id": 1, "author": "작성자", "content": "내용", "createdAt": "2025-12-17T..." }` |
